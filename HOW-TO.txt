================================================
  WHATSAPP CUSTOM WALLPAPER — HOW TO ENABLE IT
================================================

The extension files have been installed on your PC.
Follow the steps below to get it fully set up.


STEP 1 — Open the Extensions Page
------------------------------------
A tab should have opened automatically at:
  chrome://extensions

If it didn't, open Chrome and type that address
into the address bar manually.

For Edge, use:
  edge://extensions


STEP 2 — Turn On Developer Mode
---------------------------------
In the top-right corner of the extensions page,
toggle ON the switch labelled "Developer mode".


STEP 3 — Load the Extension
-----------------------------
Click the "Load unpacked" button that appears.

A folder picker will open. Navigate to and select
the folder where you installed the extension.
(The same folder you chose during installation.)


STEP 4 — PIN THE EXTENSION (important)
----------------------------------------
The extension is controlled through a popup menu.
If it isn't pinned to your toolbar, you won't be
able to open that menu at all.

In Chrome:
  1. Click the puzzle piece icon (top-right toolbar)
  2. Find "WhatsApp Custom Wallpaper" in the list
  3. Click the PIN icon next to it

In Edge:
  1. Click the puzzle piece icon (top-right toolbar)
  2. Click the EYE icon next to this extension

The extension icon will now appear permanently in
your toolbar.


STEP 5 — Open WhatsApp Web
----------------------------
Go to:
  https://web.whatsapp.com

Click the extension icon in your toolbar to open
the settings popup and start customising.


================================================
  USING THE EXTENSION
================================================

GLOBAL THEME (applies to all chats)
-------------------------------------
Click the extension icon while on WhatsApp Web.

In the popup you can:
  - Set bubble colours for outgoing/incoming messages
  - Choose an image or video wallpaper for the chat area
  - Apply blur or glass effects to backgrounds/sidebars
  - Change font family and font size
  - Tint the header and sidebar

Click "Apply Changes" to save.


PER-CHAT WALLPAPER (different per conversation)
-------------------------------------------------
  1. Open a specific chat in WhatsApp Web
  2. Click the three dots at the top-right of the chat
  3. Find the per-chat wallpaper section
  4. Set your wallpaper and apply


================================================
  TROUBLESHOOTING
================================================

Can't find the extension icon / popup won't open:
  - The extension needs to be pinned (see Step 4)
  - Click the puzzle piece icon in the toolbar
  - Pin "WhatsApp Custom Wallpaper"

Extension won't load after clicking Load unpacked:
  - Make sure you selected the correct folder
    (it must contain a file called manifest.json)
  - Make sure Developer mode is ON

Changes not showing on WhatsApp Web:
  - Click "Apply Changes" in the popup
  - Go to chrome://extensions, click Reload on
    this extension, then refresh the WhatsApp tab

Extension disappeared after a browser update:
  - Go to chrome://extensions
  - Click Load unpacked, select your folder again
  - Re-pin the extension from the toolbar

Per-chat wallpaper not applying:
  - Open the specific chat
  - Use the three-dot menu at the top of the chat
  - Re-apply from the per-chat section
